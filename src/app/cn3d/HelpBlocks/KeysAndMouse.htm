Keyboard Shortcuts

There are keyboard shortcuts for a few specific Structure Window and Sequence Window commands in Cn3D. These shortcuts may not available on all platforms, but the equivalent menu items are always present. "Ctrl/Meta" means that on some platforms, the 'control' key is used (PC and UNIX), and on others the 'meta' key is used (Macintosh).

 

Keyboard shorcuts in the Structure Window
Open Ctrl/Meta + O
Network load Ctrl/Meta + N
Save Ctrl/Meta + S
Zoom in Z
Zoom out X
Show next frame Right Arrow
Show previous frame Left Arrow
Show first frame Down Arrow
Show last frame Up Arrow
Show all frames A
Animate: play frames P
Animate: spin N
Stop animation S
Show everything E
Show aligned domains D
Select molecule M

 

Keyboard shorcuts in the Sequence Windows
Undo Ctrl + Z
Redo Shift + Ctrl + Z
Select rectangle S
Select columns C
Select blocks B
Select rows R
Horizontal drag H

 

Mouse Actions

The following tables tell how to interact with Cn3D using the mouse and keyboard together. There are two tables: one for the Structure Window, and one for the Sequence Window. "Left-drag" here means hold down the left button (or the only button, on a Macintosh) and drag the mouse. "Shift + drag" means hold down the 'shift' key while dragging the mouse.

 

Mouse actions in the Structure Window
Rotate Left-drag
Translate Shift + left-drag
Zoom Ctrl + left-drag
Select residue Double-click
Set rotation center Ctrl/Meta + double-click

 

Mouse actions in Sequence and Import Windows
Set selection Left-drag
Extend selection Shift + left-drag
Invert selection Ctrl/Meta + left-drag
Browse Entrez Left-click on title/identifier
