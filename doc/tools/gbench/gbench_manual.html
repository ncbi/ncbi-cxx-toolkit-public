Introduction

GBench is a visualization application for biomolecular structures, sequences, and sequence alignments.

Location: c++/src/gui/gbench

Running Genome Workbench

When installation is complete you can run GBench application. On Windows all you need to do is execute gbench.exe program. The program is located in Gbench Home/bin directory.

On Unix GBench required presence of certain environment variables and cannot be run as a bare Unix application. Instead you need to run run_gbench.sh shell script. Typical Unix installation includes a symbolic link or alias named gbench that can also be used.
